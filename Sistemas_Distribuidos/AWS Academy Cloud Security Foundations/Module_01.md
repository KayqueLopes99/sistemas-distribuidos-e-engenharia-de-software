## AWS Academy Cloud Security Foundations
Descrição do curso
- Módulo 1: Boas-vindas
- Módulo 2: Introdução à segurança na AWS
- Módulo 3: Proteção do acesso aos recursos da nuvem
- Módulo 4: Proteção da infraestrutura
- Módulo 5: Proteção dos dados na aplicação
- Módulo 6: Registro em log e monitoramento
- Módulo 7: Resposta e gerenciamento de um incidente
- Módulo 8: Conquista da certificação


> Depois de concluir este curso, você estará apto a:

- Identificar os benefícios e as responsabilidades de segurança ao usar a nuvem Amazon Web Services (AWS).

- Usar os recursos do gerenciamento de identidade e acesso da AWS.

- Descrever como proteger o acesso aos recursos da AWS pela rede.

- Explicar os métodos disponíveis para criptografar dados em repouso e em trânsito.

- Determinar quais serviços da AWS podem ser usados para monitoramento e resposta a incidentes.

> Documentação: https://docs.aws.amazon.com/
